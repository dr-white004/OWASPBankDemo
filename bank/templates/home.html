Welcome to the Vulnerable Banking App

This is a deliberately vulnerable Django banking application designed for educational purposes to demonstrate OWASP Top 10 vulnerabilities. It simulates a basic banking system where users can:

  • Register a new account with a starting balance of ₦100,000.00.
  • Login to access your account.
  • View account details and balance on the Dashboard (requires login).
  • Transfer Naira (₦) to other accounts (requires login).
  • Check Transaction History (requires login).
  • Logout to end your session (requires login).

Warning: This app contains intentional security flaws, including SQL Injection, Cross-Site Scripting (XSS), Cross-Site Request Forgery (CSRF), and Broken Access Control, to teach how vulnerabilities work and how to fix them. Do not use in production! Use in a controlled environment (e.g., local server or Docker) for learning purposes only.
